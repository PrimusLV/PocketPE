PocketPE is a Server Software for multiplyer Servers in the Pocket Edition of Minecraft.

PocketPE is built in PHP. This enables more flexiblity for servers and better for installing.

To install PocketPE get the installer.sh file on your server then execute it with ./install.sh and it will install and run your PocketPE Server.

There are also a few Hosting Companies who can host it for you.

We are currently in the process of re-writing the software. This means at some stages the Software might be unsable for use on commericial sevrers. Check Releases for all stable releases.
